# Cryptic Solver Visualizer

## Intent

The Cryptic Solver Visualizer is an interactive web application designed to act as a solving assistant and playback tool for cryptic novices. It consumes the puzzle metadata (`grid_state.json`) and the agent telemetry logs (`progress.jsonl`) generated by the `cryptic-solver` skill.

Its primary goals are to:
1. **Reconstruct the Puzzle Grid:** Render the true crossword layout (white cells, black cells, and clue numbers) accurately from the extracted geometry.
2. **Visualize Intermediate States:** Step through the chronological operations of the AI solver, animating the placement of partial patterns and final answers word by word.
3. **Enhance Understanding:** Provide a clean, premium interface to view clues, cross-referencing them against the current progress, and helping users understand *how* a puzzle is logically populated.

## Tech Stack
- React + TypeScript
- Vite for fast bundling
- Vanilla CSS for styling (premium, responsive, animated layout)

## Data Sources
Currently built to ingest the file snapshots found in solver workspaces (e.g., `samples/cryptic-2026-03-03`).
